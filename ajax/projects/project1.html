Mini Paint
by Brad Merrill
Project Description
This is a very simple app built using html, css and javascript that resembles the old microsoft paint of yesteryear. Nothing fancy, and one of the first projects I built using Javascript, but I still think it is a fun example of what we can do for you with basic coding. Visit the site below to try it out yourself. Choose a color from the top, single click to change to that color, and double click to revert to black.
